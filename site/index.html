What it is

Sindri Engine is a voxel-engine project inspired by sandbox worlds and mod-friendly design. The goal: a clean engine core with a fast client renderer and (later) an authoritative server.

  • Cross-platform rendering: wgpu (Vulkan/DX12/etc.)
  • Cross-platform window/input: winit
  • Compatibility flags: --fallback-gpu, --low-power

Getting Started

  1. Install Rust via rustup
  2. Clone the repo
  3. Run the client:
cargo run -p sindri_client

Fallback GPU mode (safe mode):

cargo run -p sindri_client -- --fallback-gpu

Devlog

2026-01-16 — First Render: Triangle Milestone

We created a windowed Rust client using winit, initialized wgpu, and drew our first geometry: a triangle rendered through a custom pipeline. We also added --fallback-gpu to help users with weak/broken drivers run the engine in a compatibility mode.
